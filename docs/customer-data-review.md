## Key Findings

1. **IDs are NOT globally unique**: 82 IDs exist in all 3 databases. The same ID maps to completely different customers in each factory. Merging would require ID remapping.

2. **Name overlap is minimal**: Only 16 names overlap, and of those:
   - 5 are generic/test accounts (สดปลีก, สดส่ง, etc.)
   - 3 are inter-factory transfer accounts
   - 8 are real shared customers buying from multiple factories

3. **Each factory is independent**: Customers, pricing, and bag balances are managed separately per factory. The plan to run separate POS instances per factory is the correct approach.

4. **Bag balances vary**: Some customers have large outstanding bag balances (e.g., รามอินทรา ซอย 10 at SI has 22,721 bags). Negative balances indicate over-returned bags.
